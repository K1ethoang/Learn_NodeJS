# Learn NodeJS
## Installation Guild
### 1. Clone project
```bash
git clone https://github.com/K1ethoang/Learn_NodeJS.git
```
### 2. Go to folder Learn_NodeJS
```bash
cd Learn_NodeJS
```
### 3. Install dependencies
```bash
npm install
```
### 4. Start project
```bash
npm start
```
### 5. Ctrl + Click on path "http://localhost:5000" in Terminal log
