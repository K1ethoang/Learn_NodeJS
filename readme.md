# Learn NodeJS
## Installation Guild
### 1. Clone project
`$ git clone https://github.com/K1ethoang/Learn_NodeJS.git`
### 2. Go to folder Learn_NodeJS
`$ cd Learn_NodeJS`
### 3. Install dependencies
`$ npm install`
### 4. Start project
`$ npm start`
### 5. Ctrl + Click on path "http://localhost:3000" in Terminal log
